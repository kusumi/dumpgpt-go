dumpgpt-go (v0.2.0)
========

## About

+ Parse and dump GPT in ASCII text.

+ Go version of [https://github.com/kusumi/dumpgpt](https://github.com/kusumi/dumpgpt).

## Build

Run make(1) or gmake(1).

    $ make

## Usage

    $ ./dumpgpt
    usage: ./dumpgpt: [--verbose] [--symbol] [--noalt] [-v] [-h] [-u] <gpt_image_path>
